# MEL-Arduino-101

Colección de proyectos y ejercicios de Arduino para aprender electrónica y programación de microcontroladores.

## 📚 Contenido

### Librerías
- **0_Librerias/**: Librerías necesarias
  - I2Cdev/: Librería para comunicación I2C
  - MPU6050/: Librería para acelerómetro/giroscopio MPU6050

### Ejercicios Básicos
- **ej1_Arduino_Overflow_Int/**: Desbordamiento con números enteros
- **ej1_Arduino_Overflow_Unsigned/**: Desbordamiento con números sin signo
- **ej2_Arduino_FloatType/**: Tipos de datos flotantes
- **ej2_Arduino_PinMode/**: Configuración de pines I/O
- **ej3_Arduino_AnalogRead/**: Lectura analógica
- **ej7_For/**: Bucles for

### Proyectos Intermedios
- **1_Arduino_funciones/**: Funciones en Arduino
- **ej3_LEDsPotenciometro/**: Control de LEDs con potenciómetro
- **ej4_Arduino_Servo/**: Control de servos
- **ej5_Voltimetro/**: Voltímetro digital
- **ej6_ServoPotenciometro/**: Servo controlado por potenciómetro
- **ej9_DisplayLCD/**: Pantalla LCD

### Proyectos Avanzados
- **2_5reps/**: 5 repeticiones
- **3_DCmotor/**: Motor DC
- **4_Dualservoaccel/**: Dos servos con acelerómetro
- **singleservoaccel/**: Servo con acelerómetro
- **ej8_2G/**: Aceleración 2G
- **i2c_check/**: Verificación de dispositivos I2C

### Sensores y Actuadores
- **Shock_Sensor/**: Sensor de choque
- **Speaker/**: Altavoz
- **Speaker_Choque/**: Altavoz con sensor de choque

### Documentación
- **LCD.fzz**: Esquema Fritzing de pantalla LCD
- **Arduino_Log.txt**: Registro de eventos

## 🛠️ Requisitos

- Arduino IDE o PlatformIO
- Placa Arduino (Uno, Mega, etc.)
- Librerías incluidas en `/0_Librerias`

## 📖 Instalación

1. Descarga este repositorio
2. Abre los archivos `.ino` con Arduino IDE
3. Selecciona tu placa y puerto
4. Carga el código en tu Arduino

## 📝 Notas

- Cada carpeta contiene un proyecto independiente
- Los archivos .ino son los sketches de Arduino
- Los ejercicios están organizados de menor a mayor complejidad
